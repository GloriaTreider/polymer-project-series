All the VTT examples

  • plain.html: Example that uses HTML5 to play the video and the subtitles
  • video.html: Example composed by 2 Web components (video-viewer and cue-viewer). The components work independently and they are not synchronized with each other.
  • textedvideo.html: It uses a new Web component (texted-video) that encapsulates the video-viewer and the cue-viewer. It keeps both components syncronized.
